# TCC — Teen Finance

Trabalho de Conclusão de Curso do ensino médio integrado ao técnico em **Desenvolvimento de Sistemas**, desenvolvido em 2024, na Etec prof. Marcos Uchoas Penchel, Cachoeira Paulista-SP.

## Sobre o projeto:
O **Teen Finance** é um site criado com o objetivo de auxiliar jovens na **educação financeira**, oferecendo recursos e informações para melhor controle e entendimento do dinheiro.

## Observação:
Este projeto está em sua **versão original da época**, podendo conter arquivos não utilizados ou uma organização que não reflete minhas práticas atuais.

## Tecnologias utilizadas:
- **HTML**; 
- **CSS**;  
- **JavaScript**;  
- **PHP**;  
- **MySQL** (utilizando XAMPP).

## Considerações finais:
Este projeto faz parte do meu **processo de aprendizado** como desenvolvedor e representa minha **evolução ao longo do tempo**.
